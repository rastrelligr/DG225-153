## ① สรุปเนื้อหาที่เรียน — ฉันได้เรียนรู้อะไรในสัปดาห์นี้?

= ได้เรียนรู้วิธีการ build project monogame เบื้องต้นและทำการ push เข้าไปใน github

## ② สรุปการฝึกปฏิบัติ — ฉันได้ทดลองและฝึกเขียนโปรแกรมอะไรบ้าง?

= ได้ลองเขียนและลองใช้ debug และเช็คใน terminal ได้รู้จักการ push เข้าไปในกิตเบื้องต้นและ การเรียงของโปรเจค และการทำงานเป็นขั้นตอนไม่งั้นขั้นตอนจะพลาด

## ③ ปัญหา / ข้อสงสัย — สิ่งที่ยังไม่เข้าใจหรืออยากสำรวจต่อ

= อยากสำรวจเรื่องวิธีการนำเอาไปให้คนอื่นใช้ต่อ ยังไม่เข้าใจเรื่องว่าโค้ดแต่ละอันทำงานยังไง มนัมีความสำคัญยังไง อยากรู้ที่มา +จริงๆก็ยังไม่เข้าใจภาษาโค้ดมากแต่พอมันแสดงผลแล้วเห็นภาพเราพอเข้าใจมันมากขึ้น
